# 上海彪发ERP-APP

#### 介绍

1. 下载项目：git clone https://gitee.com/wryking/shanghaibiaofa_app.git 
2. 进入项目：cd Finance 
3. 安装依赖：npm install 
4. 启动项目：npm run dev
5. 打包项目：npm run build 

#### GIT常用操作

1. 查看状态：git status
2. 拉取项目：git pull
3. 添加项目：git add .
4. 提交项目：git commit -a -m '注释内容'
5. 推送项目：git push

#### 软件架构

1. 项目基于 vue-cli 脚手架构建，使用webpack进行代码管理的vue项目
2. 金融项目：finance
3. 商品项目：goods
3. 销售项目：sale
